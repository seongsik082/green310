반복문

반복문은 말 그대로 사용자가 일일이 반복하기 힘든 일련의 동일한 동작들을 대신 처리해주는 일을 말함

  • while문 : ~하는 동안
    (조건문){실행문}: 조건이 참일 경우 동안 실행시켜라
    작성법 :
    변수를 선언합니다. var a =0; 반복조건문을 수행합니다.(a<5) {실행문 a++;}
  • for문 : ~를 위한
    for(변수지정; 조건식; 증감식){ 실행문 }

실행은 스크립트를 활용하여 제작
