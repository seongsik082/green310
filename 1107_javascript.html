조건문(if문)

실행할 스크립트 구문에 조건을 걸어 해당 조건이 참(true)이면 실행하고 거짓(false)이면 새로운 조건을 실행하게 하는 기능

작성법

if(조건식) {실행문-
조건식1이 참일때 이곳의 구문을 실행을 하고 만약 거짓이면 조건식2로 넘어감} else if(조건식) {실행문
- 조건식2가 참일때 이곳의 구문을 실행하고 만약 거짓이면 마지막 else의 실행문으로 넘어감 }
else {실행문 - 위에 조건에 모두 거짓일 때 이곳의 구문을 실행함

비교연산자

두 개 이상의 변수의 값을 서로 비교하거나 참, 거짓을 판별하는 조건식을 만들어 결과에 따라 실행시키기 위함이다.

  • a==b: a와b는 같다.
  • a!=b: a와b는 다르다.
  • a < b: a는 b보다 작다
  • a > b: a는 b보다 크다
  • a <= : a는 b보다 작거나 같다
  • a >= : a는 b보다 크거나 같다
  • 논리연산자 - 그리고(and) a조건 && b조건 : a조건 b조건 모두 만족
  • 논리연산자

실행문은 스크립트를 활용

예제문항

수학,영어,국어,과학의 점수를 prompt를 활용하여 추출하고 평균값을 계산한다. 그리고 평균이 90점 이상은 A학점/ 80점이상은 B학점 / 70점이상은 C학점/ 60점이상은 D학점/그이하는 F학점으로 표기한다.
